## 📝 Описание проекта
Данное десктопное приложение разработано на платформе WPF (Windows Presentation Foundation). Основная цель работы — реализация графического интерфейса для вычисления сложных математических выражений в соответствии с Вариантом 10.

## 🔧 Основные возможности
* **Многостраничность:** Навигация между расчетами через раздельные страницы.
* **Математический движок:** Расчет трех различных формул.
* **Валидация:** Проверка корректности ввода (обработка нечисловых данных и пустых полей).
* **Безопасность:** Обработка исключений (деление на ноль, отрицательные значения под корнем).
* **UX/UI:** Быстрая очистка полей.

### 📋 Вывод по тестированию

**Результат:** Все 4 теста пройдены успешно ✅

**Причина успешного выполнения тестов:**
В ходе выполнения практической работы были разработаны и запущены 4 автоматизированных
unit-теста методом «белого ящика» с использованием средств Microsoft Visual Studio.
Все тесты завершились успешно, поскольку логика математических функций реализована
корректно: входные данные обрабатываются без ошибок, граничные случаи (деление на ноль,
отрицательные значения под корнем) перехватываются механизмом обработки исключений,
а возвращаемые значения соответствуют математически ожидаемым результатам.

**Методы объекта Assert и их назначение:**
- `Assert.AreEqual(expected, actual)` — проверяет, что ожидаемое и фактическое значения
  совпадают; если значения различаются, тест считается проваленным.
- `Assert.IsTrue(condition)` — проверяет, что переданное условие является истинным.
- `Assert.IsFalse(condition)` — проверяет, что переданное условие является ложным.
- `Assert.ThrowsException<T>()` — проверяет, что вызов метода приводит к выбросу
  исключения указанного типа, что подтверждает корректность обработки ошибочных ситуаций.

Таким образом, тестирование подтвердило работоспособность всех трёх математических
функций приложения и корректность их поведения как при штатных входных данных,
так и в нештатных ситуациях.
